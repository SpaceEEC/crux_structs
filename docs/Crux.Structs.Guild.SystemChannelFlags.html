    Crux.Structs.Guild.SystemChannelFlags (crux_structs v0.3.0-dev) View Source

    Custom non discord API struct helping with the usage of system channel flags.

    Discord is using flags in an "inverted" way here to allow all new types to be enabled by default. Therefore all set flags are actually disabled (as their names suggest).
